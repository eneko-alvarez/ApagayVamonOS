Enunciado

Descripción de la práctica

La práctica consiste en descubrir una contraseña original a partir de un hash SHA-256, generando candidatos y comparando sus hashes hasta encontrar una coincidencia.

Parte principal

Hash

Búsqueda por fuerza bruta

El programa genera combinaciones de letras minúsculas, inserta la subcadena cr7 en distintas posiciones posibles dentro de una cadena de 7 caracteres y calcula el SHA-256 de cada candidato.

  • Lectura del hash objetivo desde el script
  • Generación sistemática de candidatos
  • Cálculo de hash con SHA-256
  • Comparación hasta encontrar coincidencia

Lógica aplicada

Python

Uso exclusivo de la primera parte del script

Para esta página se ha tomado únicamente el primer bloque del archivo Python, ignorando la segunda alternativa del script. De esta forma, la solución mostrada sigue una única estrategia de búsqueda bien delimitada.

  • Se usan solo letras minúsculas como relleno
  • La subcadena fija es cr7
  • La longitud total considerada es de 7 caracteres
  • La ejecución se detiene al encontrar la contraseña

Cada candidato se transforma en hash SHA-256. Si el resultado coincide con el hash objetivo, el programa muestra la contraseña encontrada y finaliza la búsqueda. Si no aparece ninguna coincidencia, se informa al terminar todas las pruebas.

Código

Script utilizado

Bloque usado de la práctica, mostrando solo la primera parte del archivo Python.

import hashlib
import itertools
import string

hash_objetivo = "4a00382efb5acb17c52a699364e65fc0189a4235e4fb467eb085cfa565cfeaf4"

letras = string.ascii_lowercase
subcadena = "cr7"
longitud_total = 7
longitud_restante = longitud_total - len(subcadena)

encontrado = False

for pos_subcadena in range(longitud_total - len(subcadena) + 1):
    if encontrado:
        break

    for combo in itertools.product(letras, repeat=longitud_restante):
        letras_libres = list(combo)
        candidato = letras_libres[:pos_subcadena] + list(subcadena) + letras_libres[pos_subcadena:]
        candidato_str = "".join(candidato)

        h = hashlib.sha256(candidato_str.encode()).hexdigest()

        if h == hash_objetivo:
            print(f"Contraseña encontrada: {candidato_str}")
            encontrado = True
            break

if not encontrado:
    print("No se encontró ninguna coincidencia.")
